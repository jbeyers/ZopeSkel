Check that the directory does not exist::

    >>> rmdir(tempdir, 'plone.recipe.example')

Go to a directory::

    >>> cd(tempdir)

Then use paster::

    >>> sh('paster create -t recipe plone.recipe.example --no-interactive')
    paster create -t recipe plone.recipe.example --no-interactive
    ...
